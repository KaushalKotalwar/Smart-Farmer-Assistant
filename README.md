# Smart-Farmer-Assistant
Crop disease detection web app for farmers which detects disease based on symptomms and provide chemical and Herbal solution with Fertilizer buy link
# 🌱 Smart Farmer Assistant

## 🚀 Project Overview
A web-based application that helps farmers detect crop diseases using image upload and provides solutions.

## ✨ Features
- Image-based disease detection
- Chemical & herbal solutions
- Farming tips
- Simple UI for rural users

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Python (Flask)
